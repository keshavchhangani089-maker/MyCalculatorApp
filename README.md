# MyCalculatorApp

MyCalculatorApp is a simple calculator program built in C for Windows.  
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Simple and clean console-based calculator
- Supports:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Lightweight Windows executable (`calc.exe`)

## How to Run (EXE)

1. Go to the **Releases** section or repository files.
2. Download `calc.exe`.
3. Double-click to run on a Windows machine.
